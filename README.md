# Численное интегрирование

Программа для вычисления определённых интегралов с автоматическим подбором шага (для методов Ньютона–Котеса) и с фиксированным числом узлов (для методов Гауссовского типа). Анализ методов и построение графиков ошибок.

---

## Лабораторная работа №1: Методы Ньютона–Котеса

### Реализованные методы
- **Прямоугольники** (средние) — порядок точности 2
- **Трапеции** — порядок точности 2
- **Симпсон** — порядок точности 4 (требует чётного n)
- **Три восьмых** — порядок точности 4 (требует n кратного 3)

---

## Лабораторная работа №2: Методы Гауссовского типа

### Реализованные методы
- **Гаусс** — автоматическое вычисление узлов и весов через `numpy.polynomial.legendre`, произвольное число узлов
- **Чебышев** — одинаковые веса \(A_i = 2/n\), узлы табличные для \(n = 2,3,4,5\)
- **Радо** — фиксированный левый узел \(t = -1\), узлы и веса табличные для \(n = 2,3,4,5\)
- **Лобатто** — фиксированные узлы \(t = \pm 1\), узлы и веса табличные для \(n = 2,3,4,5\)

---

## Тестовые функции (для обеих работ)
- \(x^2\) на \([0,1]\) (многочлен)
- \(e^{-x^2}\) на \([-2,2]\) (интеграл Пуассона)
- \(\sqrt{|x|}\) на \([-1,1]\) (разрыв производной)
